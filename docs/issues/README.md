# GitHub Issue Drafts

This directory now holds the 10 planned game tasks as **issue-ready GitHub drafts**.

## What is here

- `issue-01-...md` through `issue-10-...md` are the draft bodies.
- `manifest.json` is a machine-readable index with titles, labels, dependencies, and file paths.

## How to create the GitHub issues

For each issue in `manifest.json`:

1. Create a new GitHub issue using the **Issue title** from the matching markdown file.
2. Copy the markdown body below the `# Issue ...` heading into the GitHub issue body.
3. Apply the suggested labels.
4. Link dependencies in the body or as follow-up comments.
5. Assign the issue to a coding agent once its dependencies are complete.

1. Issue 1 — Set up C# project structure and solution
